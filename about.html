About Us

This dashboard was created as part of a spatial data visualization project exploring food accessibility in Seattle and King County. Our team designed the dashboard to combine geographic analysis with interactive visualization so users can explore where SNAP-authorized retailers are located and how those locations relate to neighborhood income levels. By integrating multiple datasets and building an interactive map interface, the project aims to make complex spatial data easier to understand for community members, researchers, and policymakers interested in food access and social equity.

Our Goal

The goal of this dashboard is to better understand patterns of food access for households that rely on the Supplemental Nutrition Assistance Program (SNAP). By mapping SNAP-authorized retailers alongside census income data, the dashboard highlights how food resources are distributed across neighborhoods in Seattle and King County. Through interactive exploration, users can examine whether lower-income communities have comparable access to food retailers and identify areas where access may be limited.

Methodology

This analysis combines SNAP retailer location data with income information from the American Community Survey (ACS). SNAP retailers are displayed as point features on the map, while census tract income is visualized as a choropleth layer showing variation in per-capita income across the region. The dashboard dynamically updates summary statistics based on the map view, allowing users to explore how the density of SNAP retailers changes across neighborhoods with different income levels. This interactive approach makes it possible to identify spatial patterns and potential disparities in food accessibility.

Data Sources

The dashboard integrates several publicly available datasets: - USDA SNAP Retailer Locator dataset provides locations of all SNAP-authorized food retailers. - U.S. Census Bureau American Community Survey (ACS) provides census tract level income data used to create the choropleth map. - Seattle Open Data Portal provides geographic boundary data used for spatial analysis and mapping. These datasets were processed and combined using GIS tools before being integrated into the interactive web dashboard.
